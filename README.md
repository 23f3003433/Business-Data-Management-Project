# Business-Data-Management-Project

This project is based on a gaming dataset EA SPORTS FC 25 DATABASE, RATINGS AND STATS. The objective of this project is to understand how the players overall rating is influenced by performance attributes and how players can be grouped into  meaningful segments using data driven techniques.

The dataset is being gathered from the Kaggle Platform. The link for the kaggle platform for the datatset is https://www.kaggle.com/datasets/nyagami/ea-sports-fc-25-database-ratings-and-stats

The project is being completed via secondary route
